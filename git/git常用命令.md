`git branch`                查看本地分支 有*号的为当前操作分支

`git branch -a`             查看远程分支

`git checkout branch_name`  切换分支

`git checkout -b name`      本地创建新分支并切换到新分支

`git log`                   查看git合入记录

`git clone`

`git pull`

`git status`                查看本地代码是否有待提交的代码

`git add .`

`git commit -m "提交注释" `

`git push origin branch_name`

#### 合并代码
1. git checkout master 
2. git pull
3. git branch -a
4. git merge branch_name
5. git status
6. git add file_name
7. git commit
8. git push
